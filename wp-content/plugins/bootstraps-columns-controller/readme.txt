== Description ==

With "Bootstrap's Columns Controller" you can easily use colums to customize your content

Documentation

1. To insert columns to the post/page, go to your post/page and click the button above the WP editor "Add Columns". After that, choose columns type and click "Save".

2. In editor you can easily insert the content into the columns.

3. You can remove the row click the button in the editor "Remove the row".

== Installation ==

Installing "Simple PDF Viewer" can be done either by searching for "Simple PDF Viewer" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
